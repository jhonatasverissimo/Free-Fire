<strong>Estrutura de dados Estácio - Free Fire - Simulação bolsa de itens em C </strong><br><br>

Projeto desenvolvido para praticar conceitos básicos de estrutura de dados em C, usando struct, vetores e busca sequencial.
O programa simula uma pequena mochila de jogo, permitindo cadastrar, remover, listar e buscar itens.

<strong>Objetivo</strong>
- O foco deste projeto é treinar:
- Uso de struct
- Organização de funções
- Manipulação de vetores e strings
- Implementação de busca sequencial
- Prática de lógica de programação em C

<strong>Funcionalidades</strong>
- Cadastro de até 10 itens
- Remoção de itens pelo nome
- Listagem dos itens após cada operação
- Busca sequencial por nome


<strong>Tecnologias Utilizadas</strong>
- C (C99)
- stdio.h
- string.h
